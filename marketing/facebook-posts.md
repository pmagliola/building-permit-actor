Something I've been using for solar lead gen that I don't see talked about much here — new construction building permits.

The timing is the key insight. A residential permit gets issued, then there's a 4-8 week window before the home is even framed. If you reach the builder or buyer in that window, you're having a conversation before anyone else.

Most cities publish permit data on open data portals, but it's messy and inconsistent. I built a tool that cleans it up across 36 US cities and returns it in a standard format with:
- Full address
- Permit type (new construction, addition, etc.)
- Project value
- Contractor/builder name and license

For solar specifically: filter to new construction residential above $200k project value. That's your target buyer profile.

The lead window is short — 2-6 weeks before contractors are already on site. Worth running it weekly if you're going to use it.

Roofing guys — anyone here using building permit data for prospecting? I've been playing with it and think it's an underused channel.

Here's the angle: most permit data includes the GC name. For new construction, that means you can call the builder directly instead of chasing homeowners. Builder relationships are worth 10x a homeowner lead if you can land a preferred sub relationship.

For re-roofs and renovations, some cities issue permits by type so you can filter specifically to roof work. Chicago and Raleigh are good examples — the permit type field is detailed enough to filter down to re-roof work specifically.

I built a tool that pulls this across 36 cities in a consistent format. Takes about 2 minutes to run, gives you a spreadsheet of permits from the last week/month filtered to whatever type you're targeting.

## General Construction/Trade Marketing Groups

**Post:**

Permit data as a lead source — underrated or overhyped?

I've been building tools for construction lead gen and want to share what I've learned, since I see a lot of discussion about paid ads and HomeAdvisor/Angi but not much about permit-based prospecting.

The case for it:
✅ Public record — completely legal, no grey area
✅ Includes contractor name in most cities — so you can call the GC, not just the owner
✅ Shows project value — you know if the job is worth your time before you call
✅ Timing advantage — permits issue before projects are visible anywhere else

The case against it:
❌ Data quality varies by city (some cities are months behind, some update daily)
❌ Pulling and cleaning the data takes work unless you automate it
❌ Short lead window — 2-6 weeks before you're too late

I built a scraper that covers 36 US cities and returns clean, consistent data. $1.50 per 1,000 permits — a week of Chicago permits runs about $0.50.
